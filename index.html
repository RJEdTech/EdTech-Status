Behind the scenes Infrastructure that runs the tools above — only matters if those go red too

How to use this page. Before troubleshooting a tool that isn't working, glance here. If the tile is red or yellow, the vendor is reporting a known issue — it's not just you. If the tile is green and you're still having trouble, the problem is real; status pages can lag actual outages by 30 minutes or more.

What the colors mean. Green = working. Yellow = degraded (slower or partial features). Orange = partial outage. Red = down. Slate with a wrench = our status check is broken (vendor itself may be fine; click through to verify).

Why "Behind the scenes" exists. AWS and Cloudflare run a lot of the tools above (Canvas, NoRedInk, Quizizz, and others). When something at this layer breaks, it can take tools down with it. You generally don't need to look here unless one of the tools you're using has gone red — then check whether infrastructure is also red, which means the problem may not be with the tool itself.
